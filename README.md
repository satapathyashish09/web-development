# ChronoCube - Time Management Cube

## Project Overview

ChronoCube is a responsive product landing page designed to help students and professionals manage time effectively. The website showcases features, benefits, user reviews, pricing plans, and contact information.

## Technologies Used

- HTML5
- CSS3
- Flexbox
- CSS Grid
- Responsive Design
- Tailwind CSS

## Features

- Responsive Navigation Bar
- Hero Section
- Features Section
- Benefits Section
- Testimonials
- Pricing Plans
- Footer Section
